## 10869번, 사칙연산

> 문제  
: 두 자연수 A와 B가 주어진다. 이때, A+B, A-B, A*B, A/B(몫), A%B(나머지)를 출력하는 프로그램을 작성하시오. 

 > 입력  
: 두 자연수 A와 B가 주어진다. (1 ≤ A, B ≤ 10,000)
```C++
#include <iostream>

using namespace std;

int main(int argc, char* argv[]) {
    int a, b;
    cin >> a >> b;
    cout << a+b << endl;
    cout << a-b << endl;
    cout << a*b << endl;
    cout << a%b << endl;

    return 0;
}
```
